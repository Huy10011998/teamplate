Installation

Given below are the steps you need to follow, to install the crms_laravel on your system:

Step 1: Open the terminal in your root directory(crms_laravel) & to install the composer packages run the following command:

composer install

Step 2: In the root directory, you will find a file named .env.example, rename the given file name to .env and run the following command to generate the key.

php artisan key:generate

Step 3:

  • a) Open the xampp,start the apache and mysql
  • b) Open the localhost/phpmyadmin in browser and create db_name
  • c) Update the db_name and phpmyadmin table username and password in .env file as per below screenshot

Step 4: Open the terminal in your root directory(crms_laravel) & and run the following command for migrate the table:

php artisan migrate

Step 5: Now you can run this Template in any web browser from your PC or Laptop.

LOGIN ROUTE: 
For example: localhost/crms-laravel/template/blue/public/login or www.example.com/crms_laravel/template/blue/public/login
REGISTER ROUTE: 
For example: localhost/crms-laravel/template/blue/public/registration or www.example.com/crms_laravel/template/blue/public/registration
DASHBOARD ROUTE: 
For example: localhost/crms-laravel/template/blue/public/dashboard or www.example.com/crms_laravel/template/blue/public/dashboard

File Configuration

How to change Logo ?

You can change the logo or replace it from public->assets->img folder or you can edit it from resources->views->layout->partials->header.blade.php.

<div class="navbar-header">
  <a href="index" class="navbar-brand logo">
	<img src="assets/img/logo.png" class="img-fluid" alt="Logo">
  </a>
</div>

How to change Header background color ?

You can change the header background color in style.css as per your wish

How to change Google Font ?

Go to style.css and line number 64. Here we have the default font family Poppins

@import url('https://fonts.googleapis.com/css?family=Poppins:300,400,500,700,900');
